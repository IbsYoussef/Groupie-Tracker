About

Groupie Tracker is a project that consisted of building a responsive website using a API made in JSON.
The website uses a RESTful API to showcase different artists with basic about them and show their upcoming tour dates.

Why make this?

We made this for a coding-course called the 01-Founders.

Who are the 01-Founders?

The 01-Founders is a coding school built on the self teaching model. As a student at the programme you will learn all about programming and software development.
As a student at the coding school you will study a range of programming-concepts from back-end web development to AI and machine learning.
